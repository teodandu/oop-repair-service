# oop-repair-service

C++ OOP application 

## Overview

This project implements a management system for an appliance repair service.  
It handles employees, repair requests, task allocation, real-time simulation, and reporting.

The application is built using Object-Oriented Programming principles and Modern C++ features.

## Features

- Employee management (Technicians, Receptionists, Supervisor)
- Automatic repair request allocation based on constraints
- Real-time repair simulation
- Salary calculation system with bonuses
- CSV report generation
- Error handling and input validation
- Implementation of Factory and Singleton design patterns

## Technologies Used

- C++
- STL (vector, list, queue, etc.)
- File I/O
- OOP & SOLID principles
- Design Patterns (Factory, Singleton)

## How to Run

1. Compile using a C++ compiler (C++11 or newer):
   ```bash
   g++ main.cpp -o repair_service
2. Run:
   ```bash
   ./repair_service
